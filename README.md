# Navysummer Tap

## How do I install these formulae?

`brew install navysummer/tap/<formula>`

Or `brew tap navysummer/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "navysummer/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
